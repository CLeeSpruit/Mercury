Authorization Key *:
Get an auth key from TFS

Directions

  1. Under "Personal Access Tokens", click "Add"
  2. Add a description (Mercury Task Manager)
  3. Select how long this PAT will be active. Default is 3 months. After that you'll need to create another key.
  4. Keep "All Scopes" selected
  5. Click "Create Token"
  6. Copy and paste the token into the box above. Keep this secure as you will not be able to fetch this token again.
* This is stored in your cookies. If you clear your cookies, you'll need to re-enter your key.
